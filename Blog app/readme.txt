1- Simple Blog App

A very simple blog application using HTML, CSS, and JavaScript.

2- Features
- Fetch and display posts from an API (JSONPlaceholder).
- Show posts dynamically with DOM manipulation.
- Add new posts using a simple form (client-side only).

3- How to Use
1. Open `index.html` in your browser.
2. Click **Load Posts** to fetch posts.
3. Use the form to add a new post.
