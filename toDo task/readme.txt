# My To-Do List Task

## 1. Overview
Simple JavaScript project to manage tasks in an array.

## 2. Features
Add, edit, delete, and complete tasks.

## 3. Installation
Open `index.html` in a browser.

## 4. Usage
Use console to call the functions (addTask, editTask, deleteTask, completeTask).

## 5. Technologies
JavaScript only, no DOM manipulation.
